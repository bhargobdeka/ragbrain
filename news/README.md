# Daily AI News Archive

Daily AI briefings auto-saved by RAGBrain when forwarded from the OpenClaw agent.

Each file is named `YYYY-MM-DD.md` and contains the full briefing for that day.

Files are committed manually — forward the briefing to `@RagBrain_bot` in Telegram,
then `git add news/ && git commit -m "news: YYYY-MM-DD"` when you want it public.
